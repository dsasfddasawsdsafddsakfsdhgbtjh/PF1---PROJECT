Orders & Payment

Use this page as the local clone guide for order and payment questions.

Order Status

After placing an order in a real store, customers receive a confirmation email. In this static clone, checkout is only a demo.

Payment Methods

This clone does not process real payments. Keep card, wallet, and cash-on-delivery features as display-only unless a backend is added.
